
Capsule is a plugin for Blender that augments it's in-built file exporters, allowing you to automate the export of hundreds of objects in a scene with a single click. 


F E A T U R E S
---------------
- Create reusable Export Presets that can be used across .blend files

- Supports FBX, GLTF and OBJ file exporting

- Automatically centers export targets to the scene origin (with a few caveats)

- Organise Objects and Groups with an interactive Export List

- Create multiple passes with different export parameters

- Filter objects automatically using tags

- Delicious export pie menus for your convenience ü•ß
